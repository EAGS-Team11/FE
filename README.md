# EAGS — Frontend (FE)

Project: Essay Automated Grading System (EAGS) — Frontend

This repository contains the frontend application for the EAGS project. It's built with React, Vite and TailwindCSS and integrates with a backend API for authentication and application data.

## Quick start

Prerequisites
- Node.js (v16+), npm
- Backend API running (development: `http://127.0.0.1:8000`)

Install dependencies
```pwsh
npm install
```

Run development server
```pwsh
npm run dev
```

Open the app at `http://localhost:5173` (Vite default).
